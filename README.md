# Kong E-commerce

Интернет-магазин на Next.js + Medusa.js

## Стек технологий

- **Frontend**: Next.js 16+ (App Router)
- **Backend**: Medusa.js
- **UI Kit**: Shadcn/ui + Tailwind CSS
- **Database**: PostgreSQL
- **Type Safety**: TypeScript

## Разработка

```bash
# Установка зависимостей
yarn install

# Запуск dev сервера
yarn dev
```
